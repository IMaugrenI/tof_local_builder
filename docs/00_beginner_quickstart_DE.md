# Einsteiger Schnellstart

Dies ist der kürzeste sichere Weg.

## Schnellstart

### Linux

```bash
bash scripts/start_here.sh
```

### Windows

```powershell
pwsh ./scripts/start_here.ps1
```

### macOS

```bash
./scripts/start_here.command
```

## Ablauf

1. setup (Wizard beim ersten Start)
2. start
3. check

## Ebenen verstehen

- Wizard = Erstsetup
- UI = Steuerung
- WebUI = Arbeitsfläche

## Erfolgszustand

- Stack läuft
- WebUI erreichbar
- Builder nutzbar

## Alltag

```bash
python run.py up
python run.py status
python run.py check
python run.py down
```
