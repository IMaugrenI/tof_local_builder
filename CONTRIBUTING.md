# Contributing

Thank you for your interest.

## Contribution status

This repository is currently maintained as an author-directed public frame.
General unsolicited code contributions are not accepted by default.

## Why

The repository is being kept structurally clean for future licensing control and product framing.
Unclear inbound rights can make later relicensing or commercial licensing difficult.

## Current rule

Please do not open pull requests with code changes unless you were explicitly invited to do so.

Issues, suggestions, architecture feedback, and bug reports are welcome.

## If code contributions are invited later

Any accepted code contribution may require an explicit written license grant, CLA, or equivalent contributor agreement.

## No implied rights

Discussion, issues, or review feedback do not grant trademark, endorsement, or product rights.
