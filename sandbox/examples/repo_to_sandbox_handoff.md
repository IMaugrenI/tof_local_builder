# Repo to sandbox handoff example

## Goal

Prepare a small, reviewable artifact from a read-only source.

## Source inspected

- `/workspace/source_repo_ro/...`

## Planned sandbox target

- `/workspace/builder_sandbox/output/example_handoff.md`

## Draft content pattern

```text
A) short summary
B) files inspected
C) findings
D) draft artifact path
E) exact draft content
F) risks
G) next handoff step
```

## Rule

Do not place draft files into the source repo.
Use the sandbox first.
