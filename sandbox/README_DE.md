# Sandbox

Dies ist der isolierte Schreibraum für frühe lokale Experimente.

## Zweck

Die Sandbox existiert, damit lokal unterstützte AI-Arbeit nicht direkt in echten Repos oder aktiven Stacks beginnt.

## Hier erlaubt

- Testdateien
- Entwurfsdokumente
- PlantUML-Dateien
- kleine Code-Experimente
- Scratch-Ausgaben

## Typische Pfade

- `sandbox/workspace/`
- `sandbox/output/`
- `sandbox/examples/`

## Regel

Erst hier arbeiten.
Geprüfte Ergebnisse später verschieben.
