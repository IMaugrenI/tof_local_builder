# Sandbox

Dies ist der einzige beschreibbare Bereich für überprüfte Artefakte.

## Unterordner

- `workspace/` für Entwürfe und Arbeitsnotizen
- `output/` für überprüfte Ausgaben
- `examples/` für kleine Beispiele

## Grenze

Die Quell-Repo muss read-only bleiben.
Nichts darf zurück in den gemounteten Quellpfad schreiben.
