# wizard

## Role

This space is the official first-run product wizard surface.

## Wizard responsibilities

- detect hardware
- explain recommendations
- set the source path
- set the output path
- choose light or grounded mode
- assign model slots
- save configuration
- trigger bootstrap and installation
- hand off to Open WebUI

## Product rule

There must be one official wizard truth.
Legacy helper layers may continue to exist during migration, but they should not remain equal public product realities.
